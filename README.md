# Text & Table Image Extractor

A React web application that extracts text or table data from images using OCR (Optical Character Recognition).

## Features

- Upload images via drag-and-drop or file selection
- Paste images from clipboard
- Automatic content detection (text vs table)
- Text extraction with email, phone, and URL parsing
- Table extraction with structured data output
- Export to Excel (CSV) or JSON format
- Language detection and special character preservation

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Technologies

- React 18
- Vite
- Tailwind CSS
- Tesseract.js (OCR)
- xlsx (Excel generation)
